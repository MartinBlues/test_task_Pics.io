# Pics.io Video Player – Playwright Tests

## Purpose
Automated E2E tests for Video Player feature.

## Covered flows
- video loading
- play / pause
- seeking
- fullscreen

## Why
These are P0 critical flows affecting core user experience in DAM systems.

## Run tests
npm install
npx playwright test

## Report
npx playwright show-report
